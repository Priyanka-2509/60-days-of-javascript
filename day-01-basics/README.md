Day 01 – JavaScript Basics

On Day 01, I focused on understanding the fundamentals of JavaScript and building
a strong base.

Topics covered:

Variables
- let, const, and var
- When to use each one

Data Types
- string
- number
- boolean
- Checked using typeof

Functions
- Function declaration
- Parameters and return values
- Calling functions

Arrays
- Creating arrays
- Adding elements using push() and unshift()
- Removing elements using pop() and splice()
- Creating a new array using filter()

Objects
- Creating objects using key–value pairs
- Accessing properties using dot notation
- Updating object properties
- Object methods
- Using the this keyword inside objects

Files in this folder:
- index.js – Variables, data types, functions, and array operations
- objects.js – Object creation, methods, and property manipulation

What I learned:
- How JavaScript stores and manages data
- How to group related data using objects
- Difference between mutating and non-mutating array methods
- Why objects are important for real-world applications
